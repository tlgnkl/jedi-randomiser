Джедайские практики · Рандомайзер

Честный выбор. Точные веса. Аккуратные изображения.

Генерация

Нажмите «Сгенерировать». Анимация — с замедлением, процесс — прозрачен.
Пока нет результата
Нажмите «Сгенерировать», чтобы начать.
Выбор выполняется с помощью window.crypto, затем анимация подстраивается под результат.

Практики

Редактируйте заголовки, описания, веса, блокировку и изображения. Перетаскивайте файл на карточку, чтобы загрузить картинку.
—
Изображение сжимается локально до ~1024px (качество 0.9), хранится в вашем браузере. Для синхронизации используйте «Экспорт»/«Импорт».
